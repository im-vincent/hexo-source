第二單元 常用統計量

連續型數據之特徵值:

  • 集中趨勢指標

    • 平均數
    • 中位數
    • 众数

      平均數對離群值非常敏感,而中位數或眾數則對離群值較不敏感。
      當資料中有離群值時,建議使用中位數或眾數,否則,使用平均數。

  • 分散趨勢指標

    • 全距
    • 變異數
    • 標準差

  • 偏態係數

    • 偏態係數 = 0 ,表示樣本分佈呈對稱
    • 偏態係數 > 0 ,表示樣本分佈呈右偏

    • 偏態係數 < 0 ,表示樣本分佈呈左偏

  • 峰度係數

    • 峰度係數 = 0 ,表示資料呈常態峰
    • 峰度係數 > 0 ,表示資料呈高峽峰
    • 峰度係數 < 0 ,表示資料呈低闊峰

數據特徵值之應用:

  • 經驗法則
  • 盒鬚圖
